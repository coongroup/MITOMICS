
## Running

Check that requirements are installed.
This analysis subpipeline can be invoked by running

```
Rscript make.R
```

## Requirements

    * R version 4.0.2
    * R packages:
        * drake 7.12.5
        * here 0.1
        * jsonlite 1.6.1
        * magrittr 1.5
        * qvalue 2.20.0
        * tidyverse 1.3.0
        * tsne 0.1.3
